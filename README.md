# Federated Privacy-Preserving AI Agents
A multi-level Federated Learning System built with intelligent AI agents that collaboratively train models across multiple hospitals without sharing raw data — ensuring data privacy, security, and robustness.

## Project Overview

This project implements a Privacy-Preserving Federated Learning Framework using:

🏥 Hospital AI Agents (Local Learning)

🌍 Global Aggregation Agent

🛡️ Attack Detection Agent

🎯 Strategy-Based Adaptive Training

🔁 Multi-Round Federated Orchestration

The system simulates real-world healthcare collaboration across countries while preserving sensitive patient data.

---

## Key Features
### 1. Federated Learning

- Each hospital trains locally.

- Only model updates are shared.

- Raw data never leaves local environment.
---

### 2. AI Agents (Level-Based Evolution)
## 🧠 AI Agents (Level-Based Evolution)

| Level   | Capability                                      |
|---------|------------------------------------------------|
| Level 1 | Basic federated training                       |
| Level 2 | Intelligent strategy tuning                    |
| Level 3 | Reward tracking + Adaptive decision making     |

---

###  3. Attack Detection

The system detects:

- Data poisoning attempts  
- Abnormal weight updates  
- Suspicious local behaviors  

---

###  4. Multi-Round Training

- Global model improves over rounds  
- Performance tracked each round  
- Reward-based strategy refinement
